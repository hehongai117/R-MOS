# Teaching P0 (Ghost Hand + Scaffolding + Evidence + Teacher Console) Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Deliver Phase 1 “教学闭环”功能：Ghost Hand 基础引导、4 档放权、证据引擎基础版、教师中控台 MVP、作业分发/回收、基础评分报告。

**Architecture:** 保持现有 Task/SOP 执行链不变，引入 Teaching Domain（Class/Course/Assignment/Attempt/GuidancePolicy）与 EvidenceLink 解耦。Phase 1 仅规则引导与基础证据聚合，但数据结构预留 Phase 2/3。

**Tech Stack:** FastAPI + SQLAlchemy (async) + PostgreSQL, React + TypeScript + Ant Design + Three.js + Zustand.

from sqlalchemy.ext.asyncio import create_async_engine, async_sessionmaker
from app.models.base import Base

@pytest.fixture
async def db_session():
    engine = create_async_engine("sqlite+aiosqlite:///:memory:")
    async with engine.begin() as conn:
        await conn.run_sync(Base.metadata.create_all)
    Session = async_sessionmaker(engine, expire_on_commit=False)
    async with Session() as session:
        yield session
    await engine.dispose()
